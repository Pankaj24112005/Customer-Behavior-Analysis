# Customer-Behavior-Analysis
Customer behavior analysis using 3,900+ transactions to uncover insights on spending patterns, customer segments, and product preferences. Includes data cleaning in Python, SQL-based analysis, and an interactive Power BI dashboard for business insights.

## Overview  
This project analyzes customer shopping behavior using transactional data. It demonstrates an end-to-end data analytics workflow, including data loading, cleaning, SQL analysis, dashboard creation, and reporting. The objective is to generate actionable insights for better business decisions.

## Dataset  
- ~3,900 customer purchase records  
- Includes:
  - Customer demographics (Age, Gender, Location)  
  - Purchase details (Item, Category, Amount, Season)  
  - Behavioral data (Discounts, Reviews, Frequency, Subscription)  
- Missing values handled during preprocessing  

## Tools & Technologies  
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL (PostgreSQL / MySQL / SQL Server)  
- Power BI  
- Gamma (for presentation)  
- Jupyter Notebook  

## Project Steps  

1. Data Loading  
   - Imported dataset using Python (Pandas)

2. Data Cleaning  
   - Handled missing values  
   - Standardized column names  
   - Performed feature engineering  

3. Exploratory Data Analysis (EDA)  
   - Analyzed trends and patterns  
   - Created visualizations  

4. SQL Analysis  
   - Loaded data into database  
   - Performed queries for business insights  

5. Dashboard Creation  
   - Built interactive Power BI dashboard  

6. Reporting & Presentation  
   - Created report and presentation  

## Dashboard  
The Power BI dashboard presents key metrics such as revenue, customer distribution, product performance, and purchasing trends.

## Results & Insights  
- Identified high-value customer segments  
- Analyzed the effect of discounts on spending  
- Determined top-performing products and categories  
- Compared subscriber and non-subscriber behavior  

## How to Run  

1. Clone the repository  
git clone https://github.com/your-username/customer-behavior-analysis.git

2. Install required libraries  
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2

3. Run the Jupyter Notebook for analysis  

4. Connect to your SQL database and execute queries  

5. Open the Power BI file to view the dashboard
